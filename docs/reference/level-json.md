# Level JSON Schema

## width: *Number*

Width of the Level.

## height: *Number*

Height of the Level.

## offsetX: *Number*

Offset of the Level on the X axis. Useful for loading multiple chunked Levels.

## offsetY: *Number*

Offset of the Level on the Y axis. Useful for loading multiple chunked Levels.
